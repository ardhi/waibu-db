DB Helper for [Waibu MPA](https://github.com/ardhi/waibu-mpa)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install waibu-db
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```waibu-db``` in it
